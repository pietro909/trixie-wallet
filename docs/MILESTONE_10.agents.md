# Milestone 10: Assets support

Goal: Add support for custom assets within the Arkade ecosystem.

This milestone should prove:
- Assets show in the actitives list (sent, received, minted, burned, ...)
- Assets details are shown in the activity details view (name, quantity, icon, ...)
- Assets are shown in the balance breakdown (Wallet page)
- Assets operations are supported (send, receive, mint, burn, ...)
- Feature parity with the sibling `../wallet` app

## Current State
- There are `Asset Sent` and `Asset Received` activities but they don't show asset name and icon
- No other Asset operations are supported
